Все компоненты, о которых будет написано ниже, нужно использовать ТОЛЬКО из shadcn, устанавливая через npx shadcn@latest add <component>!

1. Во всех админках нужно использовать сайдбар: https://ui.shadcn.com/docs/components/radix/sidebar. В него нужно поместить все элементы админок, чтобы был унифицированный дизайн у всего. Кроме элементов в сайдбаре должна быть кнопка выхода, как это в примере показано на сайте. Плюс все админки нужно привести к единому стилю.

2. ВЕЗДЕ, где используются поля ввода, нужно использовать https://ui.shadcn.com/docs/components/radix/input-group, на сайте написаны разные кейсы (для пароля, для текста, для строки поиска и т.д.). Для больших полей ввода использовать https://ui.shadcn.com/docs/components/radix/textarea

3. Там, где нужны чекбоксы, используем https://ui.shadcn.com/docs/components/radix/checkbox

4. Там, где нужно выбрать из выпадающего списка, используем https://ui.shadcn.com/docs/components/radix/select

5. Все кнопки должны быть реализованы через https://ui.shadcn.com/docs/components/radix/button

6. в админке sso нужно использовать такую талицу для списка пользователей https://ui.shadcn.com/docs/components/radix/data-table 

7. в админке traffic нужно добавить встроенный сканер qr кодов для добавления новых киосков. В самом куаре регистрации убрать ссылку и оставть только идентификатор, чтобы никто кроме админа не мог дойти даже до входа в админку

8. у ВСЕХ приложений должна быть поддержка светлой и тёмной темы (светлая, тёмная, системная). Параметры обеих тем берём из базовой темы shadcn (как например в админке traffic - вот такую тёмную нужно раскатать везде).

9. Внутри приложений повторяющиеся ui элементы или группы ui элементов нужно выносить для переиспользования и объединять в новые ui элементы, чтобы не таскать их по одному

10. Если какие-то компоненты ещё необходимы, изучи https://ui.shadcn.com/docs/components
